## Base URL

すべてのエンドポイントは、次の接頭辞から始まります： `{site_url}/api/v1`

## 認証{#authentication}

API キーや、 JWT Bearer トークン、有効なセッションによって認証を要するすべてのエンドポイントは、 `/auth/token` 及び `/auth/refresh` を期待します。  
[認証](../02.authentication/) をご覧ください。

## 一般的なレスポンスコード{#common-response-codes}

| コード | 説明 |
|------|-------------|
| 200 | 成功 |
| 201 | 作成 |
| 204 | コンテンツが無い（削除成功） |
| 400 | リクエスト失敗 |
| 401 | 認証されていない |
| 403 | 許可されていない |
| 404 | 存在しない |
| 409 | 衝突 (ETag が適合しない) |
| 422 | バリデーションエラー |
| 429 | レート制限 |
| 500 | サーバー内部エラー |
